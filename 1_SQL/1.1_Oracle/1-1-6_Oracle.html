1. 조인

1) INNER JOIN

SELECT * FROM [TABLE1] JOIN [TABLE2] ON [COLUMN1] = [COLUMN2]


2) LEFT OUTER JOIN

SELECT * FROM [TABLE1] LEFT OUER JOIN [TABLE2] ON [COLUMN1] = [COLUMN2]


3) RIGHT OUTER JOIN

SELECT * FROM [TABLE1] RIGHT OUTER JOIN [TABLE2] ON [COLUMN1] = [COLUMN2]


4) FULL OUTER JOIN

SELECT * FROM [TABLE1] FULL OUTER JOIN [TABLE2] ON [COLUMN1] = [COLUMN2]

2. 집합 연산자


1) UNION(합집합)

(SELECT userID FROM [TABLE1]) UNION (SELECT userID FROM [TABLE2])


2) UNION ALL(중복포함 합집합)

(SELECT userID FROM [TABLE1]) UNION ALL (SELECT userID FROM [TABLE2])


3) INTERSECT(교집합)

(SELECT userID FROM [TABLE1]) INTERSECT (SELECT userID FROM [TABLE2])


4) MINUS(차집합)

(SELECT userID FROM [TABLE1]) MINUS (SELECT userID FROM [TABLE2])

3. 제약조건

명령어 설명

NOT NULL

 해당 값의 도메인으로 NULL 값이 거부된다.

UNIQUE

 해당 값의 도메인으로 중복값이 거부된다.

CHECK(범위)

 해당 값의 도메인으로 범위 내의 값이 적용된다.

PRIMARY KEY

 해당 속성을 기본키로 적용한다.

FOREIGN KEY

 해당 속성을 외래키로 적용한다.

DEFAULT(값)

 해당 속성의 기본값을 "값" 으로 적용한다.

ALTER TABLE [테이블] ADD CONTRAINT [제약조건명] [제약조건](범위)

목차로




4. 시퀀스

명령어 설명

INCREMENT BY [숫자]

 생성된 시퀀스가 숫자만큼 증가한다.

START WITH [숫자]

 시퀀스가 숫자부터 시작한다.

NOMINVALUE , MINVALUE

 최솟값 제한을 두지 않음 , 최솟값 제한

NOMAXVALUE , MAXVALUE

 최댓값 제한을 두지 않음 , 최댓값 제한

CYCLE , NOCYCLE

 최댓값 도달시 처음부터 시작 , 최댓값 도달 시 시퀀스 중지

CACHE , NOCACHE

 시퀀스 값을 메모리에 미리 할당 , 시퀀스값을 할당하지 않음
