SQL 쿼리문 기본

기본 쿼리 목차

1. 데이터 정의어(DDL - Data Definition Language)

2. 데이터 조작어(DML - Data Manipulation Language)

3. 데이터 제어어(DCL - Data Control Language)

4. 산술 연산자

5. 비교 연산자

6. 관계 연산자









1. DDL - CREATE / ALTER / DROP

명령어 예시

CREATE

 CREATE TABLE [테이블명] ( [기본 값] [NULL허용 여부] , ... )

ALTER

 ALTER TABLE [테이블명] [RENAME TO] [CHANGE] [MODIFY] [ADD] [DROP] , ...

DROP

 DROP TABLE [테이블명] [CASCADE CONSTRAINT]
목차로







2. DML - INSERT / SELECT / UPDATA / DELETE

명령어 예시

INSERT

 INSERT INTO [테이블명] ([칼럼], ...) VALUES ([값], ...)

SELECT

 SELECT [컬럼] FROM [테이블] WHERE [조건]

UPDATE

 UPDATE [테이블명] SET [칼럼] WHERE [조건]

DELETE

 DELETE FROM [테이블명] WHERE [조건]

SELECT의 상세 활용은 쿼리 중급에서 다룹니다.

목차로







3. DCL - GRANT / REVOKE / COMMIT / ROLLBACK

명령어 예시

GRANT

 권한 참조

REVOKE

 DCL 참조

COMMIT

 COMMIT

ROLLBACK

 ROLLBACK
목차로







4. 산술 연산자

명령어 설명 예시

+ , - , * , /

 기본 산술연산자(숫자에만 적용할 수 있음) SELECT COLUMN+1 FROM TABLE;

||

 문자열 합치기 연산 'kokochi' || 'kochi'
목차로







5. 비교 연산자

명령어 설명 예시

= , ! , < , > , >= , <=

 기본 비교 연산자(문자열에도 사용가능) SELECT COLUMN FROM TABLE WHERE CONDITION = 1

!= , ^= , <>

 NOT 연산자 (모두 같은의미로 사용) SELECT COLUMN FROM TABLE WHERE CONDITION <> 1

IS NULL , IS NOT NULL

 NULL값을 체크하는 연산자 SELECT COLUMN FROM TABLE WHERE CONDITION IS NULL
목차로







6. 관계 연산자

명령어 설명 예시

AND

 && 관계연산 SELECT COLUMN FROM TABLE WHERE CONDITION = 1 AND CONDITION = 2

OR

 || 관계연산 SELECT COLUMN FROM TABLE WHERE CONDITION = 1 OR CONDITION = 2

BETWEEN

 BETWEEN 관계연산 SELECT COLUMN FROM TABLE WHERE CONDITION BETWEEN 1 AND 2

IN

 집합 내에서 조건을 찾음 SELECT COLUMN FROM TABLE WHERE CONDITION IN (1,2,3)
