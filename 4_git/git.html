깃의목적
버전관리
깃은 내가 작업하고 있는 파일들의 변경사항을 저장하고, 표시할 수 있도록 도움을 준다.
깃을 이용하면 보다 간편하게 나의 작업 사항들을 확인하고, 저장하고, 수정할 수 있으며, 이전으로 돌아가는것도 가능하다.
백업
깃은 별도의 저장소에 나의 작업한 자료들을 저장하는 방식이므로, 보다 안전하게 내 파일들을 보관할 수 있다.
또한, 작업사항들을 원격저장소에 간편하게 push하고, pull 할 수 있도록 도와준다.
협업
깃을 이용하면, 여러대의 컴퓨터에 백업된 자료들을 pull을 통해서 받아올 수 있다.
이를 이용해서 다른 사람과 자료들을 나눠가지며, 보다 간편하게 협업을 할 수 있도록 도와준다.
