스프링 4.3.X
스프링 4.0은 자바 8버전을 지원합니다. 이에 맞게 람다식, Optional 객체 등을 사용할 수 있습니다.
Groovy DSL을 사용하여 외부 빈 설정이 가능합니다.
Core 컨테이너에 추가적인 부분이 개선되었고(몇가지 어노테이션 추가, 커스텀 메타 어노테이션 정의가능 등)
WebSocket(커넥션을 연결한 상태에서 양방향 통신이 가능한 TCP/IP 기반 프로토콜), STOMP 메세징(webSocket을 사용한 메세징 프로토콜), @RestController 어노테이션 기능 추가
@Conditional을 사용하여, Configuration 내에서 조건적으로 Spring Bean을 등록할 수 있다.

4.1 : JMS, 캐싱기능 향상, RequestParam, RequestHeader 등 어노테이션에 Optional 사용가능
4.2 : @Bean이 default 메소드 감지, @Import로 어노테이션 임포트 가능
4.3 : 생성자 주입시, @Autowired키워드 생략가능, @Configuration에서 생성자 주입지원
