- Java Flight Recorder, Java Misson Control 지원
JFR : 실행중인 java 어플리케이션 진단 및 프로파일링 데이터 수집

- 도커 컨테이너 기능향상

Java10부터 JVM은 컨테이너 제어그룹에 의해 설정된 제약조건을 사용하여 메모리, CPU 제한을 설정함
