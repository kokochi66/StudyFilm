오라클 설치 및 실습환경 만들기

이 게시판은 새롭게 오라클을 시작하는 사람이 아닌, 이미 공부가 끝난 사람이 복습, 혹은 까먹은 부분을 되새기기 위해서 만들어놓은 사이트입니다.

오라클 데이터베이스가 처음이고, 배우고자 하시는 분이라면 다른 강좌를 확인해보시기 바랍니다.




1. 오라클 데이터베이스 설치하기.

뉴렉처 강의 사이트
오라클 다운로드 사이트

기본 절차 : 오라클 홈페이지에서 데이터베이스 다운로드 => 비밀번호 설정 => SQLPLUS에서 확인

현재 JDK버전이 최신인지 확인하며, 환경변수에서 자바 PATH 설정이 제대로 되어있는지 확인해야한다!




2. SQL Developer 설치하기

SQL Developer 다운로드 사이트

다운로드 후, 위의 실행창이 뜨면, JDK의 위치를 지정해준다.


SQL Developer를 실행하고, +버튼을 클릭 후에, 서버를 생성한다.
