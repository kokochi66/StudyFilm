SQL 데이터의 기본 데이터 타입정리

1. 문자형(Character) 데이터 타입

타입 정의 크기 예시

CHAR(n)

 고정길이 문자 타입-남은 값 공백으로 채움 최대2000byte 'Hello'

VARCHAR2(n)

 가변길이 문자 타입-공백으로 채우지 않음 최대4000byte 'Hello'

NCHAR(n)

 고정길이 유니코드 문자(다국어 입력 가능) - 디폴트 값 1byte 최대2000byte 'Hello'

NVARCHAR2(n)

 가변길이 유니코드 문자(다국어 입력 가능) - 디폴트 값 1byte 최대2000byte 'Hello'

LONG

 가변길이 문자 최대2GB 'Hello'

CLOB

 대용량 텍스트 최대4GB 'Hello'

NCLOB

 유니코드 대용량 텍스트 최대4GB 'Hello'



2. 숫자형(Number) 데이터 타입

타입 정의 크기 예시

NUMBER(P,S)

 십진수 기준의 가변길이 숫자 :: P와 S를 입력하지 않으면 자동지정
P(1~38) default:38 , S(-84~127) default:0		 최대22byte 1952

FLOAT(P)

 이진수 기준의 실수
P(1~128) default:22		 최대22byte 2.56

BINARY_FLOAT

 32비트 부동 소수점 수 최대4byte 5.0

BINARY_DOUBLE(n)

 32비트 부동 소수점 수 최대8byte 66.2



3. 날짜 데이터 타입

타입 정의 크기 예시

DATE

 BC4712년 1월 1일부터 9999년 12월 31일까지 입력가능
default : YYYY/MM/DD 		'2020-11-27', '23:59:59', 19981231235959

TIMESTAMP

 년,월,일,시,분,초,밀리초까지 입력가능 '2020-11-27', '23:59:59', 19981231235959
